O que é HTML?

HTML(HyperText Markup Language) é a linguagem de marcação usada para criar páginas da web. Ela define a estrutura e o conteúdo que serão exibidos em um navegador, como textos, imagens, links, tabelas e muito mais.
Diferente das linguagens de programação, o HTML não executa ações ou cálculos — ele apenas organiza e marca os elementos que formam a página. Podemos pensar no HTML como o esqueleto de um site.

Estrutura básica

Todo documento HTML começa com uma estrutura padrão, composta por tags (ou “etiquetas”) que indicam o início e o fim de cada elemento. As tags são escritas entre sinais de menor e maior (< e >).

Exemplo da estrutura básica:

    html
  • <!DOCTYPE html>
  • <html lang="pt-BR">
  • <head>
  • <meta charset="UTF-8">
  • <title>Minha Página</title>
  • </head>
  • <body>
  • <h1>Olá, mundo!</h1>
  • <p>Esta é a minha primeira página HTML.</p>
  • </body>
  • </html>

Principais partes do HTML 
<!DOCTYPE html> 👉 Informa ao navegador que o documento está escrito em HTML5. 
<html> 👉 É a tag raiz, que envolve todo o conteúdo da página. 
<head> 👉 Contém informações sobre o documento, como o título e os metadados.
<title> 👉 Define o nome que aparece na aba do navegador.
<body> 👉 É onde fica o conteúdo visível, como textos, imagens, links e listas.
Exemplos de tags comuns

<h1> a <h6> 👉 Títulos e subtítulos (do maior para o menor).
<p>** 👉 Parágrafo de texto. <a href="..."> 👉 Link para outra página. <img src="..."> 👉 Imagem. <ul>** e <li> 👉 Lista não ordenada e seus itens. <table>**, <tr>, <td> 👉 Usadas para criar tabelas.

Conclusão

O HTML é o **primeiro passo** no desenvolvimento de sites. Com ele, aprendemos a estruturar o conteúdo; depois, usamos o **CSS** para deixar o site bonito e o **JavaScript** para deixá-lo interativo. Dominar o HTML é essencial para quem deseja criar páginas web do zero e entender como a internet funciona por dentro.
